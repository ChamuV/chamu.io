Overview

This isn‚Äôt really a standalone project but rather a numerical skill I picked up while working on several PDE-based models. I first came across IMEX (Implicit‚ÄìExplicit) schemes during my pattern formation coursework and found them extremely useful for speeding up long simulations without losing stability or accuracy. I now use them regularly as part of my solver framework and plan to keep refining their implementation in future work.

IMEX schemes are time-stepping methods that split a system into stiff and non-stiff components. The stiff terms are handled implicitly to maintain numerical stability, while the non-stiff parts are treated explicitly to save computation. This balance makes IMEX schemes particularly effective for reaction‚Äìdiffusion and nonlinear diffusion problems, where fully explicit methods would require prohibitively small time steps and fully implicit ones would be computationally expensive.

Description

We consider semi-discrete systems of the form

$$\frac{du}{dt} = F(u) + G(u),$$

where \(F(u)\) is the non-stiff (explicit) term and \(G(u)\) is the stiff (implicit) part. An IMEX scheme integrates \(F\) using an explicit Adams‚ÄìBashforth (AB) method and \(G\) using an implicit multistep method such as Adams‚ÄìMoulton (AM) or Backward Differentiation Formula (BDF). The general two-step IMEX‚ÄìAB2‚ÄìAM2 scheme is:

\[ \frac{u^{n+1} - u^n}{\Delta t} = \tfrac{3}{2} F(u^n) - \tfrac{1}{2} F(u^{n-1}) + \tfrac{1}{2} \big[G(u^{n+1}) + G(u^n)\big]. \]

Here, \(F\) (explicit) can represent nonlinear reaction terms, while \(G\) (implicit) captures diffusive or stiff coupling operators that would otherwise cause numerical instability in explicit schemes. The resulting linear system for \(u^{n+1}\) is efficiently solved using pre-assembled sparse matrices (e.g. with scipy.sparse.linalg.spsolve).

Algorithm: IMEX‚ÄìAB2‚ÄìAM2

  1. Given: \(u^0, u^1\), time step $\Delta t$.
  2. For each $n = 1, 2, \dots, N-1$:
    • Compute explicit part: \(F(u^n)\) and \(F(u^{n-1})\).
    • Form RHS: \(\text{RHS} = u^n + \tfrac{\Delta t}{2}\big(3F(u^n) - F(u^{n-1}) + G(u^n)\big)\).
    • Solve implicitly: \((I - \tfrac{\Delta t}{2}A_G)u^{n+1} = \text{RHS}\), where \(A_G\) is the Jacobian or diffusion operator.
  3. Repeat until final time \(T\).

Results showed that IMEX schemes offer a robust compromise between accuracy and efficiency, maintaining stability at large $\Delta t$ values while preserving second-order temporal convergence. In nonlinear diffusion models, IMEX integration remained stable even when explicit schemes failed, confirming its suitability for biological PDE systems with mixed stiffness.
