Overview

This project started as part of an optimisation assignment. My supervisor suggested I look into the PALM algorithm ‚Äî short for Proximal Alternating Linearized Minimisation [1] ‚Äî which is used for tackling non-convex, non-smooth optimisation problems.

I thought I could make PALM run faster and had the idea of borrowing concepts from another well-known method, FISTA (Fast Iterative Shrinkage-Thresholding Algorithm) [2], which uses momentum-based acceleration. In the end, I built a kind of hybrid ‚ÄúFISTA-PALM‚Äù version that keeps the structure of PALM but adds an acceleration step to speed up convergence.

Description

This project explores how momentum-based acceleration can improve the convergence speed of Proximal Alternating Linearised Minimisation (PALM)‚Äîa method widely used for solving non-convex, non-smooth optimisation problems of the form:

$$\min_{x,\,y}\; F(x,y) = f(x) + g(y) + H(x,y),$$

where \(f\) and \(g\) are possibly non-smooth functions with simple proximal operators, and \(H(x,y)\) is a smooth coupling term with Lipschitz-continuous partial gradients. The standard PALM algorithm alternates between proximal updates on each variable block:

\[ \begin{aligned} x^{k+1} &= \operatorname{prox}_{\lambda_k f}\!\big(x^k - \lambda_k \nabla_x H(x^k, y^k)\big), \\ y^{k+1} &= \operatorname{prox}_{\mu_k g}\!\big(y^k - \mu_k \nabla_y H(x^{k+1}, y^k)\big), \end{aligned} \]

Although PALM is guaranteed to converge under mild assumptions, it often suffers from slow progress, as updates proceed independently without any momentum or extrapolation. To address this, a FISTA-inspired acceleration step was introduced, yielding a hybrid scheme‚ÄîFISTA‚ÄìPALM‚Äîthat combines PALM‚Äôs block-wise structure with Nesterov-type inertial dynamics.

At each iteration, extrapolated (‚Äúlook-ahead‚Äù) points are formed as:

\[ \tilde{x}^k = x^k + \beta_k (x^k - x^{k-1}), \quad \tilde{y}^k = y^k + \beta_k (y^k - y^{k-1}), \]

where the momentum coefficient \(\beta_k\) follows the classical Nesterov update rule:

\[ t_{k+1} = \frac{1 + \sqrt{1 + 4t_k^2}}{2}, \qquad \beta_k = \frac{t_k - 1}{t_{k+1}}. \]

Algorithm: FISTA‚ÄìPALM

  1. Initialisation: Choose starting points \(x^0, y^0\), step sizes \(\lambda_k, \mu_k > 0\), and set \(t_0 = 1\).
  2. For \(k = 0, 1, 2, \ldots\):
    • Compute momentum parameters:
      \(t_{k+1} = \frac{1 + \sqrt{1 + 4t_k^2}}{2}\), \(\beta_k = \frac{t_k - 1}{t_{k+1}}\).
    • Extrapolate:
      \(\tilde{x}^k = x^k + \beta_k (x^k - x^{k-1})\), \(\tilde{y}^k = y^k + \beta_k (y^k - y^{k-1})\).
    • Update first block:
      \(x^{k+1} = \operatorname{prox}_{\lambda_k f} \big(\tilde{x}^k - \lambda_k \nabla_x H(\tilde{x}^k, \tilde{y}^k)\big)\).
    • Update second block:
      \(y^{k+1} = \operatorname{prox}_{\mu_k g} \big(\tilde{y}^k - \mu_k \nabla_y H(x^{k+1}, \tilde{y}^k)\big)\).
    • Stop when a convergence criterion (e.g., \(\|x^{k+1} - x^k\| + \|y^{k+1} - y^k\|\) below tolerance) is met.

The hybrid algorithm achieved noticeably faster convergence than the vanilla PALM method on benchmark problems such as sparse matrix factorisation and dictionary learning. In most experiments, FISTA‚ÄìPALM reached comparable objective values in roughly half the number of iterations required by PALM, confirming that momentum-based acceleration can be successfully integrated into alternating proximal schemes.
